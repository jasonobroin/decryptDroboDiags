About Drobo Decrypt Diags {{ .Version}}

The DecryptDiags utility and webpage allows encrypted diags captured from a Drobo to be managed and decrypted

Usage

-w | -web : Create a website to managing encrypted diags running on the local PC (localhost). The default port is 8000, but this can be overloaded with -wp | -webport

-w p | -webport : Overide the default port used for the webserver

Extra info

An uploads directory will be created in the current working directory to store uploaded diags files. To access previously uploaded diag files, run from the same directory as previously.
